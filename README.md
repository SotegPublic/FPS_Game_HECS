# FPS_Game_HECS
Шутер от первого лица. Разрабатывался на основе фреймворка HECS

Проект делал в команде из 3-х человек. На текущий момент он проходит CPI тесты. Участвовал в разработке в течении года.
Отвечал за разработку логики инвентаря, убежище (мета), логику повидения дронов (сборщик, спасатель), UI в режиме миссии и режиме убежища, системы
постоянных баффов/абилок (утилиты/артефакты), систему миссий, логику лута, и прочие более мелкие фичи.
